# AI Engineering Workshop

A growing collection of self-paced exercises that cover the AI engineering stack end to end—agentic systems, generative AI, deep learning, classic machine learning, and reinforcement learning. Each exercise is designed to be small enough to finish quickly while still focusing on production-minded practices (testing, packaging, reproducibility).

## What lives here
- **Agentic AI**: patterns for tool-augmented agents, orchestrators, and evaluators.
- **Generative AI / LLMs**: prompting labs, fine-tuning workflows, retrieval, and safety guardrails.
- **Deep Learning**: model-building notebooks and training scripts (vision, NLP, multimodal).
- **Machine Learning**: feature pipelines, model serving, and MLOps hygiene.
- **Reinforcement Learning**: environment setups, policy training loops, and evaluation dashboards.
